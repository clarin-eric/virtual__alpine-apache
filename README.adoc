=== Software

. https://packer.io[Packer] 0.10.
. https://www.docker.com/[Docker] 1.11.

== To build the image

=== Enter a build environment

[source,sh]
----
env \
  IMAGE_NAME='alpine-apache' \
  IMAGE_TAG='0.3.0' \
  REGISTRY='docker.clarin.eu' \
    sh
----

=== Build the image

[source,sh]
----
sudo -E packer build 'packer/docker.json'
----
